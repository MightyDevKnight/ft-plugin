### Image

To use an image for the avatar, set the image-url and label attributes. This will take priority and be shown over initials and icons.

export const Image = args => {
  return html`
  <div style="padding: 10%; display: flex; justify-content: center; gap:16px;">
  <sux-avatar   label="a smiling girl" .size="${args.size}" image-url="https://images.unsplash.com/photo-1494790108377-be9c29b29330?ixlib=rb-1.2.1&ixid=MnwxMjA3fDB8MHxzZWFyY2h8Mnx8YXZhdGFyfGVufDB8fDB8fA%3D%3D&auto=format&fit=crop&w=800&q=60"></sux-avatar>
  </div>
  `;
};

<Canvas>
  <Story name="Image">{Image.bind()}</Story>
</Canvas>

### Initials

When you don't have an image to use, you can set the initials attribute to show something more personalized than an icon.

export const Initials = args => {
  return html`
  <div style="padding: 10%; display: flex; justify-content: center; gap:16px;">
  <sux-avatar   label="a smiling girl" .size="${args.size}" initials="SDS"></sux-avatar>
  </div>
  `;
};

<Canvas>
  <Story name="Initials">{Initials.bind()}</Story>
</Canvas>

### Icon

When no image or initials are set, an icon will be shown. The default avatar shows a generic "user" icon, but you can customize this with the icon slot.

export const Icon = args => {
  return html`
  <div style="padding: 10%; display: flex; justify-content: center; gap:16px;">
  <sux-avatar  label="a smiling girl" .size="${args.size}"></sux-avatar>
  <sux-avatar  label="a smiling girl" .size="${args.size}"><sux-icon slot="icon" name="archive-fill"></sux-icon></sux-avatar>
  </div>
  `;
};
